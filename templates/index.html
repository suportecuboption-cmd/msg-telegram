Emojis Animados

Registre emojis animados para uso automático nas mensagens

Como registrar automaticamente

Abra o chat privado com o bot no Telegram e envie uma mensagem contendo os emojis animados que deseja usar. O bot extrai o custom_emoji_id via API e salva o mapeamento automaticamente.

Nas mensagens agendadas, use o emoji normalmente no texto (modo HTML) — o sistema envolve automaticamente com <tg-emoji emoji-id="..."> antes de cada envio.
